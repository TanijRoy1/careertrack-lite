# CareerTrack Lite

A full-stack job application tracker built with React, Node.js, Express.js, TypeScript, PostgreSQL, and Prisma ORM.

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

### Backend

- Node.js
- Express.js
- TypeScript

### Database

- PostgreSQL
- Prisma ORM
